Abstract
The IJS Agroallied project is a full-stack e-commerce platform designed to revolutionize the online agricultural retail experience. It integrates Paystack payment processing, real-time admin dashboards, live chat support, and secure order management. This project demonstrates the practical application of web development, payment integration, and database management skills acquired during the PCA phase.

Background of the Study
During my PCA training, I learned how to build both client-side and server-side applications.
I developed strong skills in React for frontend development, PHP for backend scripting, and MySQL for database management. These skills were applied in the IJS Agroallied project to create a responsive, secure, and user-friendly online store. In real-world scenarios, these technologies enable developers to create scalable platforms capable of handling transactions, customer data, and analytics.

Identified Problem
Many agricultural businesses and farmers face difficulties in reaching a larger audience due to the lack of an online platform. They rely heavily on physical sales and local markets, which limits their growth potential. In addition, those who attempt online sales often lack a proper system for managing orders, specifically time-sensitive bookings like day-old chicks, payments, and customer communication. These challenges result in inefficiency, missed sales opportunities, and customer dissatisfaction.

Proposed Solution
The proposed solution was to develop a comprehensive e-commerce platform named IJS Agroallied. This platform provides the business with a fully functional online store that integrates Paystack for payments, real-time order tracking, an admin dashboard for analytics, and customer chat support. By applying the skills gained from the PCA phase, this project offers a scalable, automated, and efficient solution to the challenges faced by local agricultural businesses.

Solution
I built IJS Agroallied using Next.js for the frontend and PHP with MySQL for the backend. The frontend features an elegant, responsive interface that allows customers to browse products, add items to their cart, and make secure payments through Paystack. On the backend, I implemented user authentication, product inventory management, order processing, and payment verification. I also developed an admin panel that provides valuable insights through charts, key performance indicators (KPIs), and customer management tools.

Implementation
The implementation process began with the setup of the development environment using XAMPP and Node.js. The database schema was created in MySQL to handle user data, orders, products, and payments. The frontend was built using React components within the Next.js framework, and backend APIs were developed to handle CRUD operations and payment verification through Paystack’s API. Testing was done using Paystack’s test keys and test cards before deploying the system.

Impact
After the implementation of the IJS Agroallied platform, the simulated test environment showed improved efficiency in managing online orders and payments. Administrators could monitor sales and bookings in real-time through the dashboard, while customers enjoyed a smooth shopping experience. The system’s integration with Paystack ensured secure and verified transactions, which built trust and reliability in the platform.

Conclusion
The IJS Agroallied project successfully demonstrated how web technologies can be used to solve real-world business problems. It provided a complete solution for agricultural businesses to transition from traditional sales to online platforms. The project also enhanced my technical and problem-solving skills, especially in full-stack web development, payment system integration, and database security. Moving forward, the platform can be scaled to support more payment options and integrated with logistics APIs for delivery tracking.
